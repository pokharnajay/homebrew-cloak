# Homebrew Tap for Cloak

Invisible overlay for Claude Code and OpenAI Codex.

## Install

```bash
brew install pokharnajay/cloak/cloak
```

Or tap first:

```bash
brew tap pokharnajay/cloak
brew install cloak
```

## Update

```bash
brew upgrade cloak
```

## Uninstall

```bash
brew uninstall cloak
```
